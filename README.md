### Textual calculator • [visit](https://qagovaerts.github.io/textual_calculator/)

You can type any mathematical expression in just one textbox and the result will be calculated after a press on the ‘calculate’-button.

Note that only mathematical expressions are allowed and the expression must be written in a Pythonic syntax.
